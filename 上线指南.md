# FinCalc Lab 上线指南（引擎 A）

网站已经建好并在本地验证通过。剩下的步骤需要注册账号（涉及实名/收款），只能您本人操作。
每步都很简单，全程免费（除域名约 70-100 元/年）。**做完第 1 步就可以回来找我，后面的技术活我来。**

## 本地预览

随时看效果：在 `05-finance-site` 文件夹打开命令行，运行 `python build.py serve`，
浏览器访问 http://127.0.0.1:8788

## 第 1 步：注册 GitHub（10 分钟，最优先）

1. 打开 github.com，用邮箱注册账号。
2. 回来告诉我账号名，我帮您把网站代码推上去。
   （有了仓库，我才能给您配"每周自动更新网站"的云端定时自动化——全自动化的核心就在这一步。）

## 第 2 步：部署到 Cloudflare Pages（15 分钟，免费）

1. 打开 dash.cloudflare.com，注册账号。
2. 左侧菜单 Workers & Pages → Create → Pages → Connect to Git → 授权 GitHub → 选择网站仓库。
3. 构建设置：Framework 选 None；Build command 填 `python build.py`；Output directory 填 `dist`。
   （如果构建报错，直接把报错发给我。）
4. 部署完成后会得到一个 `xxx.pages.dev` 的免费网址，网站即刻全球可访问。

## 第 3 步：买域名（10 分钟，约 70-100 元/年）

1. 推荐在 Cloudflare 内直接买（Domain Registration → Register），成本价无溢价，且省去解析配置。
2. 域名建议：fincalclab.com 或类似（.com 优先；被注册了可加词，如 fincalclab.io/fincalcpro.com）。
3. 买好后在 Pages 项目 → Custom domains 里绑定，一键完成。
4. 告诉我最终域名，我把代码里的站点地址改掉并重新构建（影响 sitemap 和 SEO）。

## 第 4 步：Google Search Console（15 分钟）

1. 用 Google 账号打开 search.google.com/search-console。
2. 添加资源 → 输入你的域名 → 按提示验证（Cloudflare 买的域名可一键 DNS 验证）。
3. 左侧 Sitemaps → 提交 `https://你的域名/sitemap.xml`。
4. 之后 Google 开始收录。**这一步做完就进入等待期，收录通常需要 2-6 周。**

## 第 5 步：申请 AdSense（收录之后，约第 4-8 周）

1. 等 Search Console 里显示 15+ 页面被收录后再申请（过早申请易被拒，拒了有记录）。
2. 打开 google.com/adsense 注册，填入域名。
3. 它会给你一段验证代码——发给我，我加到网站里重新部署。
4. 审核通过后，我再把广告位加进模板（位置我已预留）。
5. 收款：满 100 美元起付，电汇到国内银行卡；后台按提示填 W-8BEN 税务表（人在中国选税收协定，美国预扣税降为 0）。

## 之后的全自动运转（我的工作）

仓库上线后，我给您配置 Cursor 云端定时自动化：

- **每周一次**：自动写 2-3 篇新文章/新计算器页面，构建并提交，Cloudflare 自动部署上线。
- **每周一次**：检查 Search Console 收录情况（可选，需授权）。
- 您唯一的动作：偶尔看一眼自动化的运行报告。

## 诚实的预期管理

| 时间 | 预期状态 |
| --- | --- |
| 第 1-6 周 | 建站+收录期，收入 0，属正常 |
| 第 2-4 个月 | AdSense 过审，开始有零星几美元 |
| 第 6-12 个月 | 若内容持续增长+部分页面有排名，月收入几十到几百美元 |
| 长期 | 财经类 CPC 高，起量后天花板高于一般内容站；但也存在始终不起量的可能 |

关键变量是 Google 是否给流量，这不受任何人控制。我们能控制的是：页面数量、内容质量、更新频率——这三样全部自动化了。
